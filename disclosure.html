Affiliate Disclosure

UPDATED · 2026.05.20

Adviseist 部分文章包含 affiliate link。這頁誠實說明這些 link 怎麼運作、怎麼確保不影響評測客觀性。

核心原則 我使用 affiliate link 賺取小額 commission，但這 從不影響我給的評分或推薦。如果某產品我認為不好，即使能抽佣我也會直接寫缺點。

哪些 link 會抽佣？

每篇含 affiliate link 的文章開頭會有「📌 揭露」段落明確標註。

我怎麼避免利益衝突？

  • 先評測，再考慮 affiliate：所有文章先寫完評測結論，才檢查哪些產品有 affiliate program 可加。從不為了「能抽佣」改變排名。
  • 負評照寫不修：即使是高佣金產品，缺點該寫就寫。看 心率帶評測 我給小米 ★ 3.8 就是例子。
  • 不接「廠商建議的推薦排序」：排名完全是我自己評估，廠商不能干涉。
  • 不放「最佳推薦」按鈕在每篇開頭：很多評測站把抽佣最高的產品放第一個並加大字 CTA。本站每個產品同等視覺權重。

不會抽佣的內容

  • 所有文章內鏈（連到本站其他文章）
  • Wikipedia、官方規格表、學術文獻
  • 提到的小品牌或 niche 廠商通常沒 affiliate program
  • 本站的「不推薦」段落（從不放 affiliate link）

讀者該怎麼看待 affiliate link？

說實話——如果你想支持 Adviseist 繼續寫評測，從本站的 affiliate link 購買確實有幫助（小額 commission 累積能 cover 域名年費）。但如果你在 Amazon 已經有 Prime 會員、或習慣別的購買管道，請直接用你習慣的方式買，不要為了「支持站台」改變購買行為——我不希望你為了 USD 3 的佣金多花 5 分鐘導航。

透明度承諾

本站每季公開 affiliate 收入區間（不是精確數字，是區間 USD 0-50 / 50-200 / 200-500 等），讓讀者了解規模。目前還在起步階段，2026 Q2 月收入 USD 0-30 範圍。
